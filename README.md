## 📌 Project Scope

This project performs a comprehensive **Advanced Sales Trend Analysis** using Python. The analysis covers the full data science pipeline — from raw data ingestion to an executive-level dashboard — using real-world style sales data for FY 2023.

---

## 🛠️ Tech Stack

- **Python 3.12**
- **Pandas** — Data loading, cleaning, feature engineering
- **NumPy** — Numerical computations
- **Matplotlib** — Custom visualizations & dashboards
- **Seaborn** — Heatmaps & statistical plots
- **Scikit-learn** — Polynomial regression forecasting model
